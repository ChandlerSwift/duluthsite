@swftbot: A Duluth Project

About this Bot:

@swftbot is a Python app powered by Tweepy built as a public outlet for automation information.

What can I do?

  • Tweet @swftbot and get a response (occasionally)
  • Control my light:
    • @swftbot light on — turns light on
    • @swftbot light off — turns light off
    • @swftbot light blue — turns LEDs blue (acceptable colors)
